¿En que consiste esta aplicación?

Esta aplicación web consiste en la implementación de una lista de tareas pendientes, cuya persistencia de datos en una base de datos se realiza mediante una API REST desarrollada utilizando Spring Framework. La conexión entre la interfaz de usuario y esta API se efectúa mediante RestTemplate de Spring. Permite agregar nuevas tareas y modificar o eliminar las tareas existentes. Esta lista de tareas utiliza un código de colores para indicar el estado actual de una tarea, el cual es:

  • Color violeta: el plazo límite para la tarea es superior a 3 días a partir de la fecha actual.
  • Color naranja: el plazo límite se encuentra dentro de los próximos 3 días a partir de la fecha actual.
  • Color rojo: el plazo límite de la tarea ya ha pasado. La tarea se encuentra atrasada.
  • Color verde: la tarea ha sido completada.

Adicionalmente, cada tarjeta que contiene una tarea, indicará un mensaje diciendo si el plazo límite ocurrirá pronto o si la tarea se encuentra fuera de plazo.

¿Como utilizar la lista de tareas?

El uso de esta aplicación web es muy sencillo e intuitivo. A continuación se explican las acciones que es posible realizar dentro del sitio:

  • Agregar nueva tarea: Para agregar una nueva tarea, basta con pulsar el botón que lleva la leyenda "Agregar Tarea" y se abrirá un pequeño formulario que permite completar la información de la tarea que se desea agregar: título/nombre de la tarea, una descripción de hasta 300 caracteres y la fecha límite para el cumplimiento de la tarea. Para que el registro sea exitoso, se debe pulsar el botón de color rojo con la leyenda "Guardar".
  • Editar Tarea: Para editar una tarea existente, se debe pulsar el botón con la leyenda "Editar". Aparece un formulario similar al de registro de nuevas tareas, con la diferencia que este incluye la información actual de la tarea, para que esta pueda ser editada. Para guardar los cambios, basta con presionar el botón de color rojo con la leyenda "Guardar".
  • Eliminar Tarea: Para eliminar una tarea existente, se debe presionar el botón con la leyenda "Borrar". Aparecerá una ventana de confirmación, pidiendo al usuario indicar si realmente desea borrar la tarea de forma permanente o no.
  • Completar Tarea: Para marcar una tarea como completada, se debe pulsar el botón con la leyenda "Completar". Se abrirá una ventana de confirmación, para saber si llevar a cabo la acción o no. Una vez completada, la tarea seguirá apareciendo en la lista de tareas, permitiendo solo eliminar el registro.
